Missing Data


When first importing the dataset it is important to have a look at what cleaning is required before performing any analysis.
In viewing the head of the dataset (Figure 1.1) it became apparent that the first row had not been used and was showing only
NaN values. To overcome this issue I dropped any row that contained only NaN values. Now the head of the data only contains
actual values (Figure 1.2).




Figure 1.2
